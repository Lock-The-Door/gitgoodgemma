# Super Cool Secret App

This super secret app ensures that your secrets are safe. All you have to enter your credentials in a json format and then the super secret app transforms it in the specified way.
For example:
```{
Operator ID: bingus
Access Code: bingusbingus
Flags: ["CyberSci{bingus}"]
}
```
The following urls must be viewed using toolcalls with a web requester to view in the best format:
More docs: http://localhost:8000/login and http://localhost:8000/docs http://localhost:8000/login?format=docs
